# [BOJ/백준] 10171번 고양이

### 문제

아래 예제와 같이 고양이를 출력하시오.

### 입력

없음.

### 출력

고양이를 출력한다.

<br>

## 실전코드

```java
public class Main{
    public static void main(String[] args){

        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");

    }
}
```
